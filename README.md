#  Precificação Inteligente de Hospedagens e Imóveis

Projeto de **Ciência de Dados** focado em **precificação inteligente** no contexto de hospedagem e imóveis, utilizando técnicas de tratamento e análise de dados para apoiar a definição de preços mais estratégicos.

Este projeto foi desenvolvido durante um curso da **Alura**, com foco prático em manipulação de dados reais.

---

##  Objetivo do Projeto

O objetivo deste projeto é analisar e tratar diferentes tipos de dados — **numéricos, textuais e temporais** — para gerar insights que auxiliem na precificação de imóveis e hospedagens, considerando características relevantes do dataset.

---

##  Tecnologias e Bibliotecas Utilizadas

- **Python**
- **Pandas**
- **NumPy**

---

##  Tipos de Dados Trabalhados

Durante o projeto, foram realizados tratamentos e análises nos seguintes tipos de dados:

-  **Dados Numéricos** (valores, preços, métricas)
-  **Dados Textuais** (descrições, categorias)
-  **Dados de Data** (datas de cadastro, disponibilidade, etc.)

---

##  Principais Etapas do Projeto

-  Importação e exploração dos dados  
-  Limpeza e tratamento de dados inconsistentes ou ausentes  
-  Manipulação de dados numéricos  
-  Tratamento de variáveis textuais  
-  Conversão e análise de dados temporais  
-  Preparação dos dados para análise de precificação
